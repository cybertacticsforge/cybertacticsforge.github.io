TRETC Challenge Setup Instructions

Follow these steps on apps.cyber.org to set up the challenge and launch WATCHTOWER-7.

Part A — Ubuntu VM (Target)

  1. Login to apps.cyber.org.
  2. Go to the Ubuntu Virtual Machine.
  3. On the desktop, left-click and select “Open in Terminal”.
  4. Write down the IP address after the @ symbol. Example: 10.15.81.150
  5. Option 1 (type it): enter the command below.
    6. wget https://github.com/jboyce1/ppsCTF/raw/main/challenges/TRETC/target.tar.gz
  6. Option 2 (shared clipboard):
    1. Press Ctrl + Alt + Shift to open the shared clipboard.
    2. Paste the command into the white box.
    3. Press Ctrl + Alt + Shift again to close the shared clipboard.
    4. Left-click inside the terminal window and select Paste.
  7. Press Enter. You should see target.tar.gz appear on the desktop.
  8. Double click target.tar.gz on the desktop.
  9. Left-click the folder called target and select “Extract…”
    Then click the Extract button in the top right of the window → Show Files
  10. Open the extracted target folder.
  11. Left-click in the window where deploy_target.py is located and choose “Open in Terminal”.
  12. Type the following:
    13. sudo python3 deploy_target.py
  13. In about a minute it should print: “target complete”

Part B — Kali VM (Attacker)

  1. Switch to the other machines.
  2. Select your Kali Linux VM (attacker machine).
  3. Left-click on the desktop and select Open Terminal Here.
  4. Type or copy/paste the following into the terminal:
    5. wget https://github.com/jboyce1/ppsCTF/raw/main/challenges/TRETC/WATCHTOWER-7.html
  5. Press Enter. WATCHTOWER-7.html should appear on your desktop.
  6. Double click it and start your adventure.

Tip: If a command doesn’t paste, click inside the terminal window once, then try pasting again.
